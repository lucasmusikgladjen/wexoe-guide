# Wexoe Design Tokens

## Färger

```css
/* Primära */
--wexoe-white: #FFFFFF;
--wexoe-main-blue: #11325D;
--wexoe-dark-blue: #0A1F3B;

/* Neutrala */
--wexoe-light-gray: #E8E8E8;

/* Accent */
--wexoe-action-orange: #F28C28;
--wexoe-alt-green: #10B981;  /* Används sällan */
```

## Användning

| Färg | Hex | Användning |
|------|-----|------------|
| Main Blue | `#11325D` | Rubriker, primära knappar, länkar |
| Dark Blue | `#0A1F3B` | Bakgrunder, hover-states |
| White | `#FFFFFF` | Text på mörk bakgrund, bakgrunder |
| Light Gray | `#E8E8E8` | Borders, sekundära bakgrunder |
| Action Orange | `#F28C28` | CTA-knappar, viktiga actions |
| Alt Green | `#10B981` | Framgångsmeddelanden (sällan) |

## Typsnitt

Enfold använder det typsnitt som är konfigurerat i Theme Options. Kolla live-sajten för aktuellt typsnitt.

## Breakpoints (Enfold)

```css
/* Desktop: > 989px */
/* Tablet: 768px - 989px */
/* Mobile: < 768px */
```
